# Website for Urbit

Build instructions:

1. Clone and compile [nichess-ts](https://github.com/nichessgame/nichess-ts) and [vue3-nichessboard](https://github.com/nichessgame/vue3-nichessboard)

2. Install dependencies

```
npm install
npm install path-to-nichess-ts/dist
npm install path-to-vue3-nichessboard
```
Make sure to include the dist part when installing nichess-ts.

3. Build
```
npm run build
```

4. Start dev server
```
npm run dev
```
